# 🧠 CENTRE DE COMMANDE (ORIORIS)

## 1. 🏗️ Le Métier (L'Expertise "Dur")
*C'est ton historique, la "Data" brute et tes compétences fondamentales.*
- [[Agencement]] : Règles, ergonomie, normes (ton savoir 2000-2021).
- [[Cuisine & Bain]] : Tes Formations, Expérience, parcours (Nova cuisines, Compusoft, 2020...).
- [[Technique]] : Les contraintes, la pose, les métrés.

## 2. 🤝 Le Commerce (Le Moteur)
*C'est ton savoir-faire "One Shot" et relationnel, le cœur de tes interactions client.*
- [[Techniques de Vente]] : Le "R-1", le closing, les "Killer Questions".
- [[CRM & Suivi]] : Comment on ne lâche jamais un prospect (méthode Vogica/Cuisinella).
- [[Psychologie Client]] : Lire les gens en 3 secondes.

## 3. 🤖 Le Futur (L'Accélérateur)
*C'est là où tu transfères ton expertise et ton savoir-faire dans les outils de pointe.*
- [[Intelligence Artificielle]] : Tes prompts, tes scripts Python, Andrew Ng.
- [[Automatisation]] : Faire faire le sale boulot par le code.
- [[Obsidian]] : Méta-notes sur comment tu gères ton système.

## 4. 🧠 L'OS Humain (Le Pilote)
*Ton mode d'emploi personnel pour optimiser tes capacités et éviter le crash.*
- [[TDAH & HPI]] : Tes hacks, tes rythmes, la gestion de la dopamine.
- [[Santé & Sport]] : Ton carburant physique (indispensable pour le TDAH).
- [[Journal de Bord]] : Tes réflexions quotidiennes (pour vider le buffer).
